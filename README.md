# Análisis de datos financiero – Super Store

Este proyecto presenta un análisis financiero integral para la tienda retail ficticia Super Store, con el objetivo de evaluar su desempeño económico, identificar riesgos y detectar oportunidades de mejora en la rentabilidad.

El análisis fue desarrollado utilizando SQL para responder preguntas clave de negocio en los siguientes ejes estratégicos:

* Rentabilidad general del negocio

* Comportamiento por categorías y subcategorías

* Impacto de descuentos en ventas y profit

* Análisis geográfico y regional

* Segmentación de clientes

* Eficiencia operativa (finanzas y logística)

* Evaluación del portafolio de productos

* Identificación de riesgos financieros

## Objetivo

Evaluar la salud financiera del negocio a través del análisis de ventas, profit, variabilidad y patrones temporales, con el fin de generar recomendaciones estratégicas basadas en datos. Para este proyecto, se utilizó:

* SQL 

* Power BI 

* Modelado de KPIs financieros

## Entregables

Informe ejecutivo con conclusiones y recomendaciones estratégicas

Dashboard interactivo en Power BI con:

* KPIs financieros

* Segmentaciones dinámicas

* Análisis temporal

* Visualizaciones comparativas
